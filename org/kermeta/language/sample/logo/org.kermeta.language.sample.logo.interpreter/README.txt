Instructions for configuring and running the project.
-----------------------------------------------------

Configuring this project:
--------------------------
First compile with kermeta: Right click on project.kp > Clean and Build
Then you must do a 
> mvn eclipse:eclipse

In order to achieve that you can launch the dedicated launchconfiguration provided in 
the folder eclipse_launchconfiguration.

Running this project:
---------------------
Right click on project.kp and run as Kermeta application
This will launch the application on the default sample logo file.
